## Why This Project

China has a unique digital ecosystem with platforms like WeChat, Douyin (TikTok), Xiaohongshu, and services like Alipay that are not commonly used elsewhere. This project aims to:

1. **Bridge the Gap**: Collect resources specifically for building AI agents that work with Chinese platforms
2. **Language Support**: Provide resources in both Chinese and English
3. **Best Practices**: Share proven patterns for Chinese market
4. **Community**: Build a community of Chinese AI agent developers

## 📊 Use Cases

### Case 1: WeChat Customer Service Bot

**Scenario**: E-commerce company needs 24/7 automated customer support

**Tech Stack**:
- WeChaty (WeChat integration)
- LangChain (Dialog management)
- Qwen (LLM)
- Redis (Session storage)

**Features**:
- Auto-reply FAQs
- Order inquiry
- Shipping tracking
- Human agent handoff

---

### Case 2: Xiaohongshu Content Automation

**Scenario**: Brand needs to batch generate and publish Xiaohongshu content

**Tech Stack**:
- OpenClaw (Agent framework)
- Claude Sonnet (Content generation)
- Xiaohongshu API
- Google Sheets (Data management)

**Features**:
- Generate notes from keywords
- Auto-add tags and emojis
- Scheduled publishing
- Analytics dashboard

---

### Case 3: SEO Automation for Baidu

**Scenario**: SEO team needs to optimize content for Baidu search

**Tech Stack**:
- Python + LangChain
- Baidu Search API
- Webmaster Tools API

**Features**:
- Keyword research (Baidu-specific)
- Content optimization
- Competitor analysis
- Ranking monitoring
